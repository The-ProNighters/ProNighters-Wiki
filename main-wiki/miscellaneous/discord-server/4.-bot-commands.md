# 4. Bot Commands

## Doge Bot 1000 Commands

Command list:\
/band - \[Troll] Bands a user (gives you the "Band" role)\
//blacklist-gamenights - \[Host] Blacklist a user from game nights!\
/button-make - \[Host] Make a button for game nights!\
/chased-by-dog - \[Troll] Gives you the "is being chased by dog" role\
/escape - \[Misc] Escape the band pit of doom\
/game-request - \[Unused] Request a Roblox game for game night!\
/gift - \[Event] Gift someone\
/handout - \[Misc] Handout someone a role\
/pro - \[Fun] There is 10% Chance of getting "Pro"\
/worn - \[Troll] Worns a user
