# 4. Bot Commands

## Doge Bot 1000 Commands

**Legend:** \
White - Unused Commands\
<mark style="color:red;">Red - Event Commands</mark>\
<mark style="color:green;">Green - Host Commands</mark>\
<mark style="color:yellow;">Yellow - Troll Commands</mark>\
<mark style="color:blue;">Blue - Fun Commands</mark> \
<mark style="color:$info;">Gray - Miscellaneous Commands</mark>

**Command list:**\
<mark style="color:yellow;">/band</mark> - Bands a user (gives you the "Band" role)\
<mark style="color:green;">/blacklist-gamenights</mark> - Blacklist a user from game nights!\
<mark style="color:green;">/button-make</mark> - Make a button for game nights!\
<mark style="color:yellow;">/chased-by-dog</mark> - Gives you the "is being chased by dog" role\
<mark style="color:$info;">/escape</mark> - Escape the band pit of doom\
/game-request - Request a Roblox game for game night!\
<mark style="color:red;">/gift</mark> - Gift someone\
<mark style="color:$info;">/handout</mark> - Handout someone a role\
<mark style="color:blue;">/pro</mark> - There is 10% Chance of getting "Pro"\
<mark style="color:yellow;">/worn</mark> - Worns a user
