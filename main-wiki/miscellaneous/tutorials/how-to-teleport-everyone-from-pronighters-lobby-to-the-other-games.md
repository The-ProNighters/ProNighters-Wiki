# How to Teleport Everyone from the ProNighters Lobby to Other Games

{% hint style="info" %}
You must be a Host to use this feature.
{% endhint %}

{% hint style="warning" %}
You’ll also need to request access to this feature before you can use it.
{% endhint %}

### Step 1 – Join the ProNighters Lobby

First, join the ProNighters Lobby. You can access it from the Lobby page.

<figure><img src="../../.gitbook/assets/image (22).png" alt=""><figcaption><p>The ProNighters Lobby page on Roblox</p></figcaption></figure>

### Step 2 – Locate the teleport GUI

Click the GUI labeled **"Open Teleports."** If you do not see it, it means you currently do not have access to the teleport feature.

<figure><img src="../../.gitbook/assets/image (23).png" alt=""><figcaption><p>The "Open Teleports" GUI</p></figcaption></figure>

### Step 3 – Find the Place ID of the experience

Find the Place ID for the experience by checking your browser’s address bar. For example, Piggy has the Place ID `4623386862`.

<figure><img src="../../.gitbook/assets/image (24).png" alt=""><figcaption><p>Example of a Place ID</p></figcaption></figure>

### Step 4 – Enter the Place ID into the GUI

Once you have the Place ID, paste it into the GUI and click **"Teleport."**

<figure><img src="../../.gitbook/assets/image (25).png" alt=""><figcaption><p>Teleport GUI</p></figcaption></figure>

### Step 5 – You're done!

You’ve now teleported everyone in the ProNighters Lobby to another game. If you need further assistance, feel free to ask one of our Hosts.
