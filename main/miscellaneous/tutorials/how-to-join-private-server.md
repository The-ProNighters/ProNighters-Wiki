# How to Join a Game Night with a Roblox Private Server

{% hint style="info" %}
Read more about [Private / VIP Servers](https://en.help.roblox.com/hc/en-us/articles/205345050-Private-VIP-Servers-FAQ)
{% endhint %}

You must turn on **Private Servers** for **Everyone**.

<figure><img src="../../.gitbook/assets/image (4).png" alt=""><figcaption><p>Private Server Settings</p></figcaption></figure>

If you can’t enable this option, you must add the private server owner (the **Host**) as one of your **connections** on Roblox so you can join their private servers.

### If You’re on Mobile or Console

If you’re using a mobile device or a console, you can:

* Ask the Hosts to add you as a **connection** on Roblox, or
* Ask them to add you as a **member of the private server**.

**Note:** The Host may choose to decline your request to add you as a connection.

### Finalize

When you’re ready to join a Game Night that uses a private server, you can simply join the **Host’s private server**.

If you have any questions, don’t hesitate to ask our **Hosts** for help!
