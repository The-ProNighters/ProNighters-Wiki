# 1. Type of Game Nights

Here are the types of our game nights:

### 1. **Roblox**

**Roblox Game Night** is a type of game night where we host and play a variety of Roblox games together.

### 2. **Minecraft**

**Minecraft Game Night** is a type of game night where we host a Minecraft server and play together on it.

### 3. **Discord VC**

**Discord VC Game Night** is a type of game night where we host Discord-based activity games together.

### 4. **Others**

**Other Game Nights** is a type of game night where we host games that aren’t listed above.
